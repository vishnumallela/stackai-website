# Nova — Company Website

A modern marketing-site scaffold built with **Vite + Bun**, **TanStack Router**,
**Three.js** (react-three-fiber), **Framer Motion**, **Tailwind CSS v4**, plus the
agent-focused dev tools **react-grab** and **react-doctor**.

The visual design takes inspiration from Vercel's
[Geist design system](https://vercel.com/design): minimal and high-contrast,
near-neutral surfaces, restrained color (the blue accent signals state, not
decoration), tonal hairline borders, and the **Geist Sans / Geist Mono**
typefaces. Light + dark themes ship with a no-flash toggle.

## Tech stack

| Concern        | Choice                                           |
| -------------- | ------------------------------------------------ |
| Runtime / PM   | [Bun](https://bun.sh)                            |
| Build tool     | [Vite](https://vite.dev)                         |
| Routing        | [TanStack Router](https://tanstack.com/router) (file-based) |
| 3D / WebGL     | [Three.js](https://threejs.org) via [react-three-fiber](https://r3f.docs.pmnd.rs) + [drei](https://github.com/pmndrs/drei) |

| Styling        | [Tailwind CSS v4](https://tailwindcss.com) (CSS-first tokens) |
| Design system  | [Geist](https://vercel.com/design)-inspired tokens + light/dark |
| Typography     | [Geist Sans / Mono](https://vercel.com/font) via Fontsource (self-hosted) |
| Element grab   | [react-grab](https://github.com/aidenybai/react-grab) (dev only) |
| Code health    | [react-doctor](https://github.com/millionco/react-doctor) |

## Getting started

```bash
bun install
bun run dev          # start the dev server (http://localhost:5173)
```

## Scripts

| Command            | What it does                                            |
| ------------------ | ------------------------------------------------------- |
| `bun run dev`      | Start Vite dev server (react-grab auto-loads in dev)    |
| `bun run build`    | Type-check + production build to `dist/`                |
| `bun run preview`  | Preview the production build locally                    |
| `bun run typecheck`| Run TypeScript with no emit                             |
| `bun run doctor`   | Audit React code health with react-doctor               |

## Using react-grab

In development, hover any element in the running app and press **⌘C / Ctrl+C** to
copy its exact source location (file + component) — then paste into your coding
agent (Claude Code, Cursor, etc.) for faster, more accurate edits. It is loaded
only when `import.meta.env.DEV` is true, so it never ships to production.


## Design tokens

All theming lives in [src/styles.css](src/styles.css) using Tailwind v4's
CSS-first `@theme`. Semantic tokens (`background`, `subtle`, `foreground`,
`muted`, `faint`, `accent`) are defined once and flipped between light (`:root`)
and dark (`.dark`) — so utilities like `bg-background` / `text-muted` respond to
the theme automatically. Borders default to a tonal hairline; `border-strong`
steps it up. Radii map to `rounded-sm` (6px), `rounded-md` (12px), `rounded-lg`
(16px).

Add a page by creating a file under `src/routes/` — the plugin regenerates
`routeTree.gen.ts` automatically while the dev server runs.
